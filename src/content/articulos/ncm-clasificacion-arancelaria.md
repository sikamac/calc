# Código NCM: Clasificación arancelaria completa

El **Código NCM** (Nomenclatura Común del Mercosur) es el sistema de clasificación de mercaderías más importante para el comercio exterior en Argentina. Una clasificación incorrecta puede generar multas, demoras y problemas legales.

## ¿Qué es el NCM?

El NCM es un sistema de 8 dígitos que clasifica todas las mercaderías que se comercian internacionalmente en el Mercosur (Argentina, Brasil, Paraguay, Uruguay).

### Estructura del código:
```
84.71.30.00
│  │  │  └─ Subpartida (2 dígitos)
│  │  └──── Partida (2 dígitos)
│  └─────── Capítulo (2 dígitos)
└───────── Sección (2 dígitos)
```

## ¿Por qué es Importante la Clasificación Correcta?

### Consecuencias de un error:
- ❌ **Multas:** Hasta 100% del valor de los derechos
- ❌ **Demoras:** Retención de la mercadería
- ❌ **Perdida de beneficios:** No poder acceder a regimenes especiales
- ❌ **Problemas legales:** Posibles juicios con Aduana

### Beneficios de hacerlo bien:
- ✅ **Costos exactos:** Sabés cuánto vas a pagar antes de importar
- ✅ **Beneficios arancelarios:** Acceso a reducciones por origen
- ✅ **Proceso fluido:** Despacho rápido sin inconvenientes
- ✅ **Compliance:** Cumplimiento normativo

## Cómo Encontrar tu Código NCM

### Método 1: Buscador oficial ARCA

1. Entrá a [Buscador NCM ARCA](https://www.arca.gob.ar/adicuad/NCMArancel/)
2. Escribí una descripción en español
3. Navegá el árbol de clasificaciones
4. Verificá la nota explicativa del capítulo

### Método 2: Por similitud

Si ya importaste productos similares:
1. Revisá tu DJAI anterior
2. Verificá el código usado
3. Confirmá que sigue vigente

### Método 3: Asesoramiento profesional

Consultá con:
- Despachante de aduanas matriculado

## Ejemplos prácticos de clasificación

### Ejemplo 1: Laptop para uso personal

**Descripción:** Computadora portátil, marca Dell, modelo XPS 13

**Búsqueda:** "computadora portátil" → Capítulo 84

**Código resultante:** **8471.30.11**

```
84.71: Máquinas automáticas para tratamiento de datos
  .30: Portátiles (laptops)
    .11: Con peso inferior a 10 kg
```

**Arancel:** 0% (Exento por tratado de tecnología)

### Ejemplo 2: Remera de Algodón

**Descripción:** Remera 100% algodón, tamaño M

**Búsqueda:** "prendas de vestir de algodón" → Sección XI

**Código resultante:** **6109.10.00**

```
61.09: Camisetas, jerséis y chalecos
  .10: De algodón
    .00: Sin especificar más
```

**Arancel:** 35% (Protección a industria textil local)

### Ejemplo 3: Motor eléctrico industrial

**Descripción:** Motor trifásico, 5 HP, para maquinaria industrial

**Búsqueda:** "motores eléctricos" → Capítulo 85

**Código resultante:** **8501.52.00**

```
85.01: Motores y generadores eléctricos
  .52: Motores de corriente alterna, polifásicos
    .00: De potencia superior a 750W pero inferior a 75 kW
```

**Arancel:** 14% (General para maquinaria)

## Criterios de Clasificación

### 1. **Características del Producto**
- Material de fabricación
- Función/principio de funcionamiento
- Grado de elaboración

### 2. **Uso previsto**
- ¿Para qué se va a usar principalmente?
- ¿El uso determina las características esenciales?

### 3. **Notas explicativas**
Cada capítulo tiene notas que excluyen o incluyen productos específicos.

#### Ejemplo de nota excluyente:
> "Este capítulo no comprende los artículos de la partida 95.03 (juguetes)"

## Casos complicados y criterios especiales

### Caso 1: Producto multifunción

**Ejemplo:** Impresora multifunción (imprime, escanea, copia)

**Regla:** Clasificar según la función principal

**Código:** **8443.31.11** (Impresoras multifunción)

### Caso 2: Conjuntos de Artículos

**Ejemplo:** Set de herramientas con destornilladores, martillo, pinzas

**Regla:** Clasificar según el componente que da el carácter esencial

**Código:** **8205.90.00** (Sets de herramientas)

### Caso 3: Productos incompletos

**Ejemplo:** Celular sin batería

**Regla:** Clasificar como producto completo si tiene características esenciales

**Código:** **8517.12.00** (Teléfonos móviles)

## Errores comunes que debes evitar

### ❌ Error 1: Clasificar por el nombre comercial
```
Producto: "iPhone"
Error: Buscar "iPhone" en el buscador
Correcto: "Teléfono móvil" → 8517.12.00
```

### ❌ Error 2: Ignorar las Notas del Capítulo
```
Producto: Partes de repuesto
Error: No leer notas de exclusión
Correcto: Verificar si las partes están incluidas o excluidas
```

### ❌ Error 3: No considerar el estado del producto
```
Producto: Ropa usada
Error: Clasificar como ropa nueva
Correcto: 6309.00.00 (Prendas usadas)
```

## Documentación que debes tener

Para justificar tu clasificación NCM:

1. **Descripción técnica** del producto
2. **Fotografías** claras
3. **Manuales técnicos** (si aplica)
4. **Composición** (materiales, ingredientes)
5. **Uso principal** del producto
6. **Brochure comercial** o especificaciones

## Caso de Estudio: Importación de Café

**Producto:** Café tostado en grano, origen Brasil

**Proceso de clasificación:**

1. **Búsqueda inicial:** "café" → Capítulo 09
2. **Verificación:** 09.01 (Café sin tostar) vs 09.02 (Café tostado)
3. **Origen:** Brasil (Mercosur) → Beneficio arancelario
4. **Código final:** **0901.21.00** (Café tostado, sin descafeinar)

**Resultado:**
- Arancel: 0% (origen Mercosur)
- IVA: 21%
- Costo total: Valor CIF + 21% IVA
